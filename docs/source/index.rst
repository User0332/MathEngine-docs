Documentation for the MathEngine library
========================================================================

MathEngine is a series of libraries built for Mathematical Computation in C#/.NET. It functions similar to a system like Wolfram Mathematica, allowing for exact mathematical calculations and evaluations as well as floating-point approximations.

.. note::

   This project is under active development.
